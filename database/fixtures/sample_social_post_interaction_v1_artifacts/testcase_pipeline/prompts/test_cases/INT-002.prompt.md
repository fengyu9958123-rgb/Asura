#### PRD 原文
## 2. 动态流（时间线）
**操作路径：** 首页 - 推荐/关注

### 2.1 列表与排序
- **排序切换：** 支持按“最新”和“热度”两种方式切换查看。
  - **热度规则：** 热度值 = 点赞数 + 评论数 × 2，按此值降序排列。
- **分页加载：** 列表仅支持下拉加载更多（即下拉操作触发加载下一页）。
- **关注tab空态：** 当关注tab下无内容时，展示引导文案“暂无关注动态，去看看推荐”，用户点击后可跳转至推荐tab。
- **通用空态：** 列表无数据时展示占位提示，并显示“去发帖”引导。

### 2.2 卡片展示
- 展示用户头像、昵称、发布时间。
- 展示帖子正文内容。
- 图片以九宫格预览形式展示。
- 展示该帖子的点赞数与评论数。

#### PRD 原文
## 3. 点赞互动
**操作路径：** 动态卡片/帖子详情 - 点赞按钮

### 3.1 登录校验
- 需登录后才可进行点赞/取消点赞操作。未登录用户点击点赞按钮时，弹出登录引导。

### 3.2 交互与防抖
- **即时响应：** 点击后本地状态立即切换（点赞/取消点赞），点赞计数即时更新（+1/-1）。
- **防抖机制：** 本地状态切换后，异步请求未返回之前，忽略后续对该按钮的点击操作。
- **异常处理：** 异步落库失败时，回滚本地状态至操作前，并弹出提示告知用户。
