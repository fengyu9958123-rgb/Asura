## 2. 动态流（时间线）
**操作路径：** 首页 - 推荐/关注

### 2.1 列表与排序
- **排序切换：** 支持按“最新”和“热度”两种方式切换查看。
  - **热度规则：** 热度值 = 点赞数 + 评论数 × 2，按此值降序排列。
- **分页加载：** 列表仅支持下拉加载更多（即下拉操作触发加载下一页）。
- **关注tab空态：** 当关注tab下无内容时，展示引导文案“暂无关注动态，去看看推荐”，用户点击后可跳转至推荐tab。
- **通用空态：** 列表无数据时展示占位提示，并显示“去发帖”引导。

### 2.2 卡片展示
- 展示用户头像、昵称、发布时间。
- 展示帖子正文内容。
- 图片以九宫格预览形式展示。
- 展示该帖子的点赞数与评论数。

#### PRD 原文
## 5. 关注/取消关注与个人主页
**操作路径：** 动态卡片头像/昵称 - 进入个人主页

### 5.1 个人主页展示
- 展示用户头像、昵称、签名。
- **发帖列表：**
  - 支持分页加载，通过上拉加载更多。
  - 按发布时间倒序排列（新发布的在上）。
  - 无帖子时展示“TA还没有发布内容”的空态提示。

### 5.2 关注操作
- 支持关注与取消关注。操作后，粉丝/关注相关的计数即时更新。
- **交互与异常：** 点击后本地状态与计数先更新。若异步请求失败，仅弹出提示告知用户，本地状态不回滚，后续刷新页面时数据将自动纠正。
- **登录校验：** 未登录用户点击关注按钮时，弹出登录引导。
