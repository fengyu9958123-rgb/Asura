# 需求版本：社交发帖互动 v1.0
# 需求设备范围：Web/H5 前台

## 1. 发帖（图文）
**操作路径：** 发现 - 发帖

### 1.1 内容输入
- 支持文本输入，上限 1000 字。字数按中文字数计算，一个汉字计 1 字，英文单词按实际长度计。
- 支持图片上传，上限 9 张，单张大小不超过 5MB。支持拖拽与多选方式选择图片。
- **图片上传时机：** 选择图片后立即上传至服务器。
- 发布前可预览已选图片，并支持删除。删除图片的同时调用接口删除服务器对应文件。
- **必填校验：** 文本内容与图片至少其一存在，否则不可发布。

### 1.2 发布与异常
- 发布成功后，页面自动跳转至该帖子详情页。
- 发布失败时，弹出 toast 提示具体失败原因，页面停留在当前发帖页不跳转。

## 4. 评论与回复
**操作路径：** 帖子详情 - 评论区

### 4.1 评论发送
- 支持发表一级评论与二级回复（对评论的回复）。
- 输入内容长度上限 500 字。
- 新评论/回复发送成功后，页面滚动定位至该条新内容处。
- **列表排序：** 评论列表按发布时间正序排列，最早评论显示在最上方，新评论显示在底部。

### 4.2 评论删除与折叠
- 作者可以删除自己发布的一级评论/二级回复，删除前需进行二次确认。
- 当一级评论被删除后，其下的所有子回复统一折叠，并显示“评论已删除”标识。
- 用户可以点击“评论已删除”区域，展开查看被折叠的子回复内容。
